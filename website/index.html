Execution model

Intent enters once. The runtime makes every step inspectable.

01

Plan first.

Every workflow can be planned without executing commands, so operators can inspect the exact steps before commit.

02

Approve mutation.

Mutating workflows fail unless `MICROCLOUD_AGENT_APPROVAL=approved` is explicitly present.

03

Execute remotely.

Remote command wrappers let the runtime stay in Docker while the host retains the real infrastructure binaries.

04

Return structured results.

Health, workflow steps, chat responses, and execution payloads come back as operator-usable JSON.
